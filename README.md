# graphql-lsc

## Configuration

### graphql-lsc.commandWithArgs

You must specify how to boot the language server like following:

```json
{
    "graphql-lsc.commandWithArgs": [
        "docker-compose", "run", "--rm", "app"
    ]
}
```
